Ferramentas Utilizadas

É possível utilizar qualquer editor de texto simples para desenvolver uma página web, como Bloco de Notas, Notepad++, Sublime, entre outros.

A ferramenta utilizada no curso foi o VSC (Visual Studio Code) que é open source (gratuito), compatível com vários sistemas operacionais. O link para download está nas Referências.

Uma das funcionalidades que é interessante realizar a instalação é o Live Server, que irá auxiliar no desenvolvimento. Outro pluging que iremos utilizar mas já vem instalado é o Emmet, que auxilia na produtividade do código.

Estrutura Básica do HTML

A estrutura do HTML é formada por 3 partes:

  • html
  • head
  • body

O que está no corpo é o que irá aparecer para o usuário. Já a cabeça será o título da página.

Falando sobre Tags

As tags ficam entre os sinais de menor "<" e o sinal de maior ">". A tag é fechada com o sinal "/".

Algumas das Tags utilizadas:

  • i - itálico
  • u - sublinhado
  • strong - negrito
  • mark - marca texto
  • ol - lista ordenada
  • ul - lista não ordenada
  • li - tag filha

(Voltar)

Atributos das Tags

Os atributos são propriedades das tags, dividas entre genéricas e específicas.

Alguns do atributos genéricos são:

  • id - serve para o JS identificar o elemento
  • style - voltado para o CSS
  • class - também voltado para o CSS

Alguns dos atributos específicos são:

  • type - utilizado com a tag input
  • src - utilizado com a tag img
  • width - também utilizado com a tag img
  • href - específico para a tag a

Textos

"Texto é basicamente o que o usuário vai ler, ele vai entrar no site e ler textos"

As principais tipografias utilizadas para os textos são:

  • Os h's - h1, h2, h3, h4, h5, h6
  • p - utilizado para criar parágrafo
  • blockquote - utilizado para criar citação
  • sup e sub

Outros exemplos foram listados em Falando Sobre Tags

Listas Ordenadas e Não Ordenadas

Podemos criar listar utilizando as tags já mencionadas anteriormente. Para Listas Ordenadas devemos utilizar a Tag "ol". Para as Listas Não Ordenadas utilizamos a Tag "ul".

Para ambas as listas devemos usar a Tag filha "li" que irá listar os itens.

(Voltar)

Links

"Quando clicamos em links, estamos navegando na internet. A palavra link é navegação.

Para criarmos links de navegação utilizamos a Tag "a". A letra "a" remete à palavra âncora, pois estamos ancorando em outras páginas.

Essa Tag possui alguns atributos específicos, como "href" e "target".
